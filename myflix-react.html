Tech Stack

Frontend

  • React Router (for navigation and route protection)
  • React Bootstrap & Bootstrap 5 (for responsive design)
  • SCSS (for custom styling)

Deployment
  • Netlify (frontend hosting)
  • Render or Heroku (backend/API hosting)

Description

myFlix is a modern, client-side web application that empowers users to explore a vast movie database, curate their own favorites, and manage their personal profiles. Built with React, the app features a sleek and responsive design inspired by streaming platforms like Netflix, making it intuitive on both desktop and mobile devices.

  • Users can create accounts, securely log in, and personalize their movie experience.
  • The app fetches movie data from a custom-built REST API and supports profile management.
  • Fast, secure, and visually appealing UI for movie enthusiasts.

Features

  • User Authentication
    Sign up and log in using secure JWT authentication.
  • Browse Movies
    See a complete list of movies from the API.
  • Movie Details
    Access in-depth info for each movie, including genre, director, and synopsis.
  • Favorites Management
    Add or remove movies from your personal favorites list.
  • Profile Management
    Update your user profile or delete your account entirely.
  • Responsive Design
    Optimized for all devices with React Bootstrap and custom SCSS.
  • Movie Search
    Instantly search for titles using a Netflix-style search bar.
  • Protected Routes
    Only authenticated users can access certain app features (via React Router).

Highlights

  • Full-Stack Integration
    Seamless interaction between the React frontend and a Node.js/Express/MongoDB backend.
  • Modern Authentication
    Utilizes JWT for secure session management.
  • Cloud Deployment
    Live demo accessible on Netlify, with scalable backend hosting.
  • Custom Theming
    Stylish, brand-inspired UI using SCSS and Bootstrap.
  • Mobile-First Experience
    Responsive layouts and mobile navigation for users on any device.
